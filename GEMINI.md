## 🚀 Project Overview

**ZedISQL** is a high-performance, professional-grade PostgreSQL GUI client built using a modern stack:
- **Core**: Rust (Tauri)
- **Frontend**: TypeScript, Vanilla CSS
- **Database**: SQLx with asynchronous PostgreSQL pooling

## ✨ Key Features Implemented

- **Dynamic Multi-Tab Interface**: Isolated query sessions with robust tab management (New, Switch, Close).
- **Real-time Dashboard Metrics**: Live monitoring of active/idle sessions and server-wide Transactions Per Second (TPS).
- **Dynamic Object Explorer**: Hierarchical database browser with on-demand table fetching and auto-expansion.
- **Smart Result Rendering**: Support for complex PostgreSQL types (Timestamps with Chrono, Decimals, BigInts).
- **Optimized UX**:
    - F5 / Cmd+R hotkeys for contextual execution and refresh.
    - Connection auto-popup on startup.
    - Professional dark mode theme with glassmorphism elements.
